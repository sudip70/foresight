Project story

              

                A full-stack ML financial literacy product built for beginner-friendly market education.
              

            

          


          

            

              
Foresight is a full-stack market intelligence and financial literacy app.

              

                The app combines live market index history, market data refresh jobs, FastAPI
                inference routes, ML/RL allocation artifacts, scenario forecasting, backtests,
                explainability, and a polished frontend into one learning-oriented product.
              

            

            

          


          

            

              
System architecture

              

                

                  01
                  
Data ingestion

                  
    
                    
  • yfinance OHLCV for equities, ETFs, crypto, and market indices
    • 
                    
  • Configured universes from config/asset_universe.v1.json and config/market_indices.v1.json
    • 
                    
  • Macro CSV inputs for model context
    • 
                  

                

                

                  02
                  
Refresh and storage

                  
    
                    
  • GitHub Actions runs offline/supabase_refresh.py for daily Supabase updates
    • 
                    
  • Local rebuild path writes processed .npy artifacts for artifact fallback
    • 
                    
  • Market index snapshots refresh at backend startup when enabled
    • 
                  

                

                

                  03
                  
FastAPI model layer

                  
    
                    
  • Supabase forecast engine serves stored and on-demand forecasts
    • 
                    
  • Artifact engine supports ticker forecasts, simulation, RL allocation, and backtests
    • 
                    
  • Index endpoints serve latest cards and historical chart series
    • 
                  

                

                

                  04
                  
Learning frontend

                  
    
                    
  • Static HTML/CSS/JS app calls the API directly
    • 
                    
  • Market overview combines index history, scenario rankings, and sentiment
    • 
                    
  • Learn Mode turns outputs into plain-language finance lessons
    • 
                  

                

              

              

                

                  Runtime contract
                  

                    The FastAPI app prefers Supabase when SUPABASE_URL and
                    SUPABASE_SERVICE_ROLE_KEY are configured, then falls back to local model
                    artifacts so the dashboard still works offline.
                  

                

                

                  User-facing APIs
                  

                    /api/market/indices, /api/market/indices/{symbol}/history,
                    /api/forecasts/ticker, /api/forecasts/market, and portfolio simulation
                    routes power the visible workflows.
